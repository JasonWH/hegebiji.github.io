职场人士的核心竞争力，聊聊主动学习（二）

发表于 2019-09-28 | 分类于 创业

在上一篇文章中我们聊了高效率主动学习的三个要点，今天我们继续来探讨高效率主动学习的方法。想快速掌握一项技能就要先对目标有个基本认识，再利用这些信息描绘出你所要学习的范围，并且筛选出哪些书籍或资料可以用来辅助自己学习，然后建立你自己的学习计划。当你知道了学习什么和如何学习并且制定了详细的学习计划后，最后就是要执行计划，这是最重要的部分，如果没有执行，你就不能真正地掌握一项技能。

我们可以发现主动学习的三个要点可以总结出高效率学习的四个步骤：

  1. 对要掌握的技能有一个基本的了解；
  2. 通过了解的信息，确定你尚未掌握的部分，也就是确定要学习的范围；
  3. 制定有明确里程碑，可以阶段性检验的学习计划；
  4. 坚决并持续地执行学习计划。

这是一种通过确定目标，聚焦学习范围，关注重点内容的快速学习方法，这里我们可以称它为“四步学习法”。它能帮助你通过自主探索、主动获取的方式将真正需要的知识快速掌握。接下来我们会用几天时间来聊聊这个快速学习方法。你可以根据你的实际情况对这个方法进行变更和优化，去掉或变更不适合你的部分，保留或优化对你有效的部分。最后，你将总结出一套完全适合自己的主动学习方法，也对你的职业生涯会有很大的帮助。

通过分析我们可以发现，在四步学习法中，前三个步骤是一次性的工作，第四个步骤也就是执行计划部分是循环往复的。这里先说说第一个步骤，对目标的基本了解。当你开始学习某项技能的时候，往往对到底要学习什么并不是很了解，这就是人们常说的，不知道自己不知道。你首先要从宏观角度来了解它，知道学习这项技能所涉及的知识有多少。然后你要分析哪些知识是你已经掌握的，哪些知识是你没有掌握的。这样你就知道了你不知道的部分。举例来说，你想要学习短视频的制作，你可以在线上搜索与此相关的内容，浏览相关的文章和博客，这样你可以在几个小时之内就了解关于短视频制作的全局知识，并且你会知道哪些是你需要学习的，哪些是你已经掌握的知识或技能。

第一个步骤就是对你想要掌握的技能做一些基础调研。你可以通过线上搜索来完成大部分的调研工作。也可以去图书馆借阅相关的书籍，只需要关注概述性章节即可。在这个步骤中不需要在阅读资料上花费大量时间，因为第一个步骤的目的不是要掌握技能，而是对这个技能有一个宏观的了解。

鉴于四步学习法中的第二个步骤涉及了确定学习范围、定义学习目标、收集学习资源等很多内容，所以第二步的内容我们就在明天继续探讨。
